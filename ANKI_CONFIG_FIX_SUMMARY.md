### 使用方法：

1. **重新打包插件**
   ```bash
   python3 build_addon.py
   ```

2. **安装插件**
   - 在 Anki 中卸载旧版本
   - 安装新的 .ankiaddon 文件
   - 重启 Anki

3. **配置插件**
   - 工具 → 插件 → 选择 "Anki AI Chat Tool" → 配置
   - 或者使用工具菜单中的配置选项

4. **开始使用**
   - 打开任意卡片
   - 点击 "🤖 Ask AI" 按钮
   - 享受 AI 聊天功能
